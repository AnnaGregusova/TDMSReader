Project description

This Java library is designed for reading and processing of data stored in Technical Data Management Streaming (TDMS) format.

Structure

Project is divided into 11 classes

  • DataReader: This class provides methods for reading and processing different data types.
  • MetaDataReader: This class is responsible for reading and parsing file's meta data.
  • LeadInDataReader: This class is responsible for reading and parsing file's lead in data.
  • RawDataReader: This class is responsible for reading and parsing file's raw data.
  • MetaData: This class represents the information and structure of files's meta data.
  • LeadInData: This class represents the information and structure of file's lead in data.
  • TDMSSegment: This class represents a segment in a TDMS file, encapsulating the lead-in data and metadata for that segment.
  • TDMSGroup: This class represents a group within a TDMS file, containing channels and their associated properties.
  • TDMSChannel: This class represents a channel in a TDMS file, including its properties and raw data.
  • TDMSProperty: This class represents a property of a TDMS object, including its name, value, and data type.
  • TDMSFile: This class represents a TDMS file, providing methods to read and organize its contents into groups and channels.

Installation

You can find the project here: https://github.com/AnnaGregusova/TDMSReader. To run it you need to clone it using this command:

git clone https://github.com/AnnaGregusova/TDMSReader

After cloning use this command to go to the right repo:

cd TDMSReader

Now you can use maven to start the project

mvn clean install
mvn exec:java

To generate documentation use the command:

mvn javadoc:javadoc

Using the library

If you run the project, the first thing you will see is a question about which path (file) you want to execute.

Usage

1. Initializing TDMS File:

TDMSFile tdmsFile = TDMSFile.read(path);

Reads the TDMS file from the specified path and initializes a TDMSFile object.

2. Working with File Properties:

tdmsFile.getProperties();

Returns ArrayList of TDMSProperty objects(properties) associated with the TDMS file.

3. Accessing Groups:

ArrayList<TDMSGroup> groups = tdmsFile.getGroups();

Returns an ArrayList of TDMSGroup objects of all groups in the file.

TDMSGroup tdmsGroup = tdmsFile.getGroup("Group_name");

Returns a specific TDMSGroup by name.

4. Group Details and Channels:

tdmsGroup.getName();

Returns the name of the group.

tdmsGroup.getProperties();

Returns an ArrayList of properties associated with the group.

tdmsGroup.getChannels();

Returns an ArrayList containing all TDMSChannels within the group.

TDMSChannel tdmsChannel = tdmsGroup.getChannel("Channel_name");

Obtains a specific channel by name from the group.

5. Channel Information and Data:

tdmsChannel.getName();

Returns the name of the channel.

tdmsChannel.getProperties();

Returns an ArrayList of properties associated with the channel.

tdmsChannel.getPropertyValue("Property_name");

Returns the value of a specified property of the channel.

tdmsChannel.getRawData();

Returns all raw data from the channel.

tdmsChannel.getRawData(10);
